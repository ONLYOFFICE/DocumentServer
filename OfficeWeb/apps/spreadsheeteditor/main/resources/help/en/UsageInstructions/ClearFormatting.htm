Clear text, format in a cell

You can quickly remove the text or the format within the selected cell. To do that,

  1. select a cell, a range of cells with the mouse or the whole worksheet by pressing the Ctrl+A key combination,
  2. click the Clear icon at the top toolbar and select one of the available options:
    • use the All option if you wish to remove all including text, format, function etc;
    • use the Text option if you wish to remove the text from the selected range of cells;
    • use the Format option if you wish clear the format of the selected range of cells. The text and functions, if present, will remain.

Note: to clear the selected range of cells completely it's also possible to use the Clear All option from the right-click menu.
