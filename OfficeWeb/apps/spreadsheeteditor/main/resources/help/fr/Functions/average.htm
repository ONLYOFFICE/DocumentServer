Fonction AVERAGE

La fonction AVERAGE est l'une des fonctions statistiques. Elle est utilisée pour analyser la plage de données et trouver la valeur moyenne.

La syntaxe de la fonction AVERAGE est :

AVERAGE(number1, number2, ...)

où number1(2) est une valeur numérique entrée manuellement ou incluse dans la cellule à laquelle il est fait référence.

Pour appliquer la fonction AVERAGE,

  1. sélectionnez la cellule dans laquelle vous souhaitez afficher le résultat,
  2. cliquez sur l'icône Insérer une fonction située sur la barre d'outils supérieure,
    ou cliquez avec le bouton droit sur la cellule sélectionnée et sélectionnez l'option Insérer une fonction depuis le menu,
    ou cliquez sur l'icône sutuée sur la barre de formule,
  3. sélectionnez la groupe fonctions Statistiques depuis la liste,
  4. cliquez sur la fonction AVERAGE,
  5. saisissez les arguments nécessaires en les séparant par des virgules,

    Remarque: Vous pouvez entrer jusqu'à 30 valeurs numériques.

  6. appuyez sur la touche Entrée.

Le résultat sera affiché dans la cellule sélectionnée.
