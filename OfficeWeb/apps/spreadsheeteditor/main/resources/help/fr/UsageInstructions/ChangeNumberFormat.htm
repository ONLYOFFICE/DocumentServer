Modifier le format de nombre

Vous pouvez facilement modifier le format de nombre, c'est à dire l’apparence d’un nombre saisi dans votre feuille de calcul. Pour le faire,

  1. sélectionnez une cellule, une plage de cellules avec la souris ou la feuille de calcul entière en utilisant la combinaison de touches Ctrl+A,
  2. cliquez sur le bouton Format de nombre sutué sur la barre d'outils supérieure et sélectionnez le format de nombre à appliquer :
    • Général sert à afficher les données saisies en tant que nombres standards de façon la plus compacte, sans aucun signe supplémentaire,
    • Nombre sert à afficher les nombres avec 0-30 chiffres après le point décimal où un séparateur de milliers ajouté entre chaque groupe de trois chiffres avant le point décimal,
    • Entier sert à afficher les nombres sous la forme d'un nombre entier,
    • Scientifique (exponentiel) sert à exprimer le nombre sous la forme d.dddE+ddd ou d.dddE-ddd où chaque d c'est un chiffre de 0 à 9,
    • Devise sert à inclure un des symboles monétaires suivants : $ Dollar, € Euro, £ Livre, p. Rouble, ¥ Yen, et deux décimales,

      Remarque : pour ajouter vite un symbole de dollar $ aux données sélectionnées, vous pouvez également utiliser l'icône Style monétaire sutuée sur la barre d'outils supérieure.

    • Date sert à afficher les dates. Vous pouvez sélectionner l'un des formats de date suivants : MM-dd-yy, MM-dd-yyyy, dd-MM-yy, dd-MM-yyyy, dd-MMM-yyyy, dd-MMM, MMM-yy,
    • Heure sert à afficher l'heure. Vous pouvez sélectionner l'un des formats d'heure suivants : HH:mm, HH:MM:ss, hh:mm tt, hh:mm:ss tt,
    • Pourcentage sert à afficher les données en pourcentage avec un symbole de pourcentage %,

      Remarque : pour ajouter vite un symbole de pourcentage à vos données vous pouvez également utiliser l'icône Pourcentage sutuée sur la barre d'outils supérieure.

    • Texte sert à afficher les valeurs numériques qui sont traitées comme du texte simple avec la plus grande précision disponible.
  3. modifiez le nombre de décimales, si nécessaire :
    • utilisez l'icône Ajouter une décimale sutuée sur la barre d'outils supérieure pour afficher plus de chiffres après le signe décimal,
    • utilisez l'icône Réduire les décimales sutuée sur la barre d'outils supérieure pour afficher moins de chiffres après le signe décimal.
