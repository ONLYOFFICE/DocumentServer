Сортировка и фильтрация данных

Сортировка данных

Данные в электронной таблице можно быстро отсортировать, используя одну из доступных опций:

  • По возрастанию используется для сортировки данных в порядке возрастания - от A до Я по алфавиту или от наименьшего значения к наибольшему для числовых данных.
  • По убыванию используется для сортировки данных в порядке убывания - от Я до A по алфавиту или от наибольшего значения к наименьшему для числовых данных.

Для сортировки данных:

  1. выделите диапазон ячеек, который требуется отсортировать,
  2. щелкните по значку Сортировка от А до Я , расположенному на верхней панели инструментов, для сортировки данных в порядке возрастания,
    ИЛИ
    щелкните по значку Сортировка от Я до А , расположенному на верхней панели инструментов, для сортировки данных в порядке убывания.

Данные также можно сортировать, используя команду Сортировать из контекстного меню.

Фильтрация данных

Чтобы отображать только те строки, которые соответствуют определенным критериям, воспользуйтесь опцией Фильтр. Для включения фильтра:

  1. выделите диапазон ячеек, содержащих данные, которые требуется отсортировать,
  2. щелкните по значку Фильтр , расположенному на верхней панели инструментов.

В первой ячейке каждого столбца выделенного диапазона ячеек появится кнопка со стрелкой . Это означает, что фильтр включен.

Чтобы применить фильтр, нажмите на кнопку со стрелкой . В открывшемся окне Фильтр настройте параметры фильтра одним из следующих способов:

  • Выбор данных, которые надо отображать

    Снимите флажки рядом с данными, которые требуется скрыть. Для удобства все данные в окне Фильтр отсортированы в порядке возрастания. Чтобы облегчить этот процесс, используйте поле поиска. Введите в этом поле свой запрос полностью или частично и нажмите клавишу Enter. Чтобы очистить поле поиска, используйте значок .

    В случае необходимости можно задать порядок сортировки данных, которые требуется отображать, с помощью кнопки для сортировки данных в порядке возрастания, или с помощью кнопки для сортировки в порядке убывания.

  • Применение пользовательского фильтра

    Нажмите на кнопку Пользовательский в правом верхнем углу окна Фильтр. В открывшемся окне Пользовательский фильтр выберите из верхнего выпадающего списка один из доступных критериев и введите нужное значение в поле справа.

    Для добавления еще одного критерия выберите переключатель И, если требуется, чтобы данные удовлетворяли обоим критериям, или используйте переключатель Или, если могут удовлетворяться один или оба критерия. Затем выберите из нижнего выпадающего списка второй критерий и введите нужное значение справа.

В первой ячейке столбца появится кнопка Фильтр . Это означает, что фильтр применен.

Форматирование по шаблону таблицы

Чтобы облегчить работу с данными, в ONLYOFFICE™ Spreadsheet Editor предусмотрена возможность применения к выделенному диапазону ячеек шаблона таблицы с автоматическим включением фильтра. Для этого:

  1. выделите диапазон ячеек, которые требуется отформатировать,
  2. щелкните по значку Форматировать как шаблон таблицы , расположенному на верхней панели инструментов,
  3. в галерее выберите требуемый шаблон,
  4. в открывшемся всплывающем окне проверьте диапазон ячеек, которые требуется отформатировать как таблицу,
  5. установите флажок Заголовок, если требуется, чтобы заголовки таблицы входили в выделенный диапазон ячеек; в противном случае строка заголовка будет добавлена наверху, в то время как выделенный диапазон ячеек сместится на одну строку вниз,
  6. нажмите кнопку OK, чтобы применить выбранный шаблон.

Шаблон будет применен к выделенному диапазону ячеек, и Вы сможете редактировать заголовки таблицы и применять фильтр для работы с данными.

Очистка фильтра

Для очистки фильтра:

  1. выделите диапазон ячеек, которые содержат отфильтрованные данные,
  2. щелкните по значку Фильтр , расположенному на верхней панели инструментов.
