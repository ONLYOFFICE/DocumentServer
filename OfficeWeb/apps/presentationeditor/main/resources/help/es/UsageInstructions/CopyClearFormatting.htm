Copie/limpie formato

Para copiar el cierto formato de texto,

  1. seleccione el pasaje de texto con el formato deseado con el ratón o usando el teclado,
  2. pulse el icono Copiar estilo en la barra de herramientas superior,
  3. seleccione el pasaje de texto al que usted quiere aplicar el formato.

Para rápidamente eliminar formato que ha aplicado usted a un pasaje de texto,

  1. seleccione el pasaje de texto con el formato no deseado,
  2. pulse el icono Limpiar estilo en la barra de herramientas superior.
